How to play
  • Each tile has a shape (fixed color) and one colored edge on a single side. Tap a tile in your rack to select it, then rotate to choose which side the edge faces.
  • Tap a glowing board cell to place the tile.
  • Shape match: if the new tile's shape matches an adjacent tile's shape, that neighbor is removed. +1 per tile removed.
  • Edge match: if a colored edge touches a neighboring tile whose shape-color matches that edge color, the two tiles swap places. +1 point. Swaps can chain onward, but a pair won't flip back and forth forever.
  • Mutual edge match: if both tiles' edges match each other's shape-colors (e.g. a red edge facing a green shape, and that tile's green edge facing back at the red shape), the swap scores +2 instead of +1.
  • Post-swap shape match: after a swap, if either swapped tile now sits next to a stationary tile of the same shape (not its swap partner), that stationary tile is removed. +1 per tile removed.
  • Double edge match: if more than one side matches at once, nothing swaps — instead you score x2 (2 points) for each matching side.
  • The game ends when the board is full, or when you have no tiles left to draw or place.
